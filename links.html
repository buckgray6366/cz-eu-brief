06Google Search Console — do this (needs your Google login)

One Domain property covers all 8 geos at once.

  1. Open search.google.com/search-console → Add property → Domain → enter trycoolizi.com.
  2. Copy the TXT record it gives you → add in Cloudflare DNS (Type TXT, Name @) → Verify.
  3. Sitemaps → submit:
    Sitemap
    https://trycoolizi.com/sitemap.xml
  4. URL Inspection → paste each landing page → Request Indexing (the big lever for brand terms):
    Inspect / request indexing
